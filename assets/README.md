# Assets

This directory contains branding and visual assets for Splax OS.

## Structure

```
assets/
├── logo/           # Logo files
│   ├── logo.png    # Primary logo
│   ├── logo.svg    # Vector version
│   └── icon.png    # App icon (square)
├── banners/        # Social media banners
└── screenshots/    # Screenshots for documentation
```

## Usage Guidelines

- Use the SVG version when possible for best quality
- The logo should have adequate padding when displayed
- Do not distort or modify the logo colors
